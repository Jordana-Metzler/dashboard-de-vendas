# Dashboard Analítico de Vendas Globais

Este projeto consiste em um dashboard analítico desenvolvido para visualizar e analisar dados de vendas globais.  
O objetivo é responder perguntas estratégicas sobre vendas, categorias de produtos, países, descontos e valor médio vendido.

## Objetivo do Dashboard

O dashboard permite ao usuário acompanhar indicadores importantes de vendas e realizar análises por diferentes dimensões, como ano, segmento e país.

## Perguntas respondidas

O dashboard responde às seguintes perguntas:

1. Qual o valor total vendido?
2. Quantas vendas foram realizadas por categoria de produto?
3. Quantas vendas foram realizadas por país considerando a prioridade de entrega?
4. Qual foi a média de desconto nas vendas por subcategoria de produto?
5. Quais países tiveram maior média de valor de venda?  
   Essa análise é demonstrada em um mapa.

## Filtros disponíveis

O usuário pode filtrar os dados por:

- Ano
- Segmento
- País

## Indicadores e visualizações

O dashboard apresenta:

- Valor total vendido
- Quantidade de vendas por categoria
- Vendas por país e prioridade de entrega
- Média de desconto por subcategoria
- Mapa com os países de maior média de valor de venda

## Tecnologias utilizadas

- Power BI
- Base de dados de vendas globais
- Visualizações interativas
- Mapa geográfico

## Finalidade
Este dashboard foi desenvolvido como parte do curso Microsoft Power BI para Business Intelligence e Ciência de Dados, com objetivo de aprender a ferramenta e com foco na criação de uma solução visual para análise de vendas globais, facilitando a interpretação dos dados e a tomada de decisão.
